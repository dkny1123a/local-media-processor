## 🔍 架构问题分析

### 问题 1: 单体架构限制扩展性

**问题描述**:
所有处理功能集中在单一 FastAPI 应用中，难以独立扩展和部署。

**具体表现**:
- 所有处理器在同一个进程中运行
- 模型加载占用大量内存（Whisper large-v3 ~2.9GB, Demucs ~1GB）
- 无法根据负载独立扩展特定处理功能

**影响**:
- 内存占用高（预计 5-8GB）
- 无法水平扩展
- 单点故障风险

---

### 问题 2: 模型缓存缺乏管理

**问题描述**:
虽然各处理器有模型缓存机制，但缺乏统一的内存管理和释放策略。

**具体表现**:
```python
# whisper_processor.py
_model_cache = {}

def get_model(model_name="large-v3"):
    if model_name not in _model_cache:
        _model_cache[model_name] = whisper.load_model(model_name)
    return _model_cache[model_name]
```

**影响**:
- 模型常驻内存，无法释放
- 多模型同时加载可能导致内存溢出
- 缺乏模型生命周期管理

---

### 问题 3: 文件处理缺乏队列机制

**问题描述**:
文件处理直接在 API 端点中执行，缺乏任务队列和异步处理机制。

**具体表现**:
```python
# main.py
@app.post("/api/audio/process")
async def process_audio_endpoint(...):
    result = process_audio(input_path, output_path, ...)
    return {"success": result["success"], ...}
```

**影响**:
- 大文件处理阻塞 API 响应
- 无法处理并发请求
- 缺乏任务进度跟踪

---

### 问题 4: 缺乏监控和指标

**问题描述**:
没有性能监控、处理统计和健康检查机制。

**具体表现**:
- 只有基本的 HTTP 健康检查
- 没有处理时间统计
- 没有资源使用监控
- 没有错误率统计

---

### 问题 5: 文件路径管理复杂

**问题描述**:
主机路径和容器路径映射逻辑复杂，容易出错。

**具体表现**:
```python
# main.py
def resolve_host_path(path):
    """将主机路径解析为容器内可访问的路径"""
    if path.startswith('/hostroot'):
        path = path.replace('/hostroot', '', 1)
    return path
```

**影响**:
- 路径解析逻辑复杂
- 容易出现路径错误
- 跨平台兼容性问题

---

### 问题 6: 缺乏错误处理和恢复机制

**问题描述**:
处理失败时缺乏详细的错误信息和恢复机制。

**具体表现**:
```python
# audio_processor.py
except Exception as e:
    return {
        "success": False,
        "message": f"音频处理失败: {str(e)}"
    }
```

**影响**:
- 错误信息不够详细
- 缺乏错误分类和恢复策略
- 用户体验差

### 建议 1: 引入任务队列

**文件**: `backend/task_queue.py`（新建）

```python
from celery import Celery
import os

# Celery 配置
celery_app = Celery(
    'media_processor',
    broker=os.environ.get('CELERY_BROKER', 'redis://localhost:6379/0'),
    backend=os.environ.get('CELERY_BACKEND', 'redis://localhost:6379/1')
)

celery_app.conf.update(
    task_serializer='json',
    accept_content=['json'],
    result_serializer='json',
    timezone='UTC',
    enable_utc=True,
    task_track_started=True,
    task_time_limit=3600,  # 1小时超时
    worker_prefetch_multiplier=1,  # 一次只处理一个任务
)

# 任务定义
@celery_app.task(bind=True)
def process_audio_task(self, input_path, output_path, options):
    """异步音频处理任务"""
    try:
        # 更新任务状态
        self.update_state(
            state='PROCESSING',
            meta={'progress': 0, 'stage': 'loading'}
        )

        # 执行处理
        result = process_audio(input_path, output_path, **options)

        # 更新进度
        self.update_state(
            state='PROCESSING',
            meta={'progress': 100, 'stage': 'completed'}
        )

        return result
    except Exception as e:
        self.update_state(
            state='FAILED',
            meta={'error': str(e)}
        )
        raise

@celery_app.task(bind=True)
def process_image_task(self, input_path, output_path, operation, options):
    """异步图像处理任务"""
    # 类似实现
    pass
```

**改造 API 端点**:
```python
# main.py
@app.post("/api/audio/process")
async def process_audio_endpoint(...):
    # 提交任务到队列
    task = process_audio_task.delay(
        input_path,
        output_path,
        {
            "silence_threshold": silence_threshold,
            "min_silence_duration": min_silence_duration,
            "max_volume": max_volume
        }
    )

    return {
        "task_id": task.id,
        "status": "PENDING",
        "message": "任务已提交"
    }

@app.get("/api/tasks/{task_id}")
async def get_task_status(task_id: str):
    """查询任务状态"""
    task = process_audio_task.AsyncResult(task_id)

    if task.state == 'PENDING':
        return {"status": "PENDING", "message": "任务等待处理"}
    elif task.state == 'PROCESSING':
        return {
            "status": "PROCESSING",
            "progress": task.info.get('progress', 0),
            "stage": task.info.get('stage', '')
        }
    elif task.state == 'SUCCESS':
        return {"status": "SUCCESS", "result": task.result}
    else:  # FAILED
        return {"status": "FAILED", "error": task.info.get('error', '')}
```

```python
import torch
import psutil
import time
from typing import Dict, Optional
import threading

class ModelManager:
    """统一模型管理器"""

    def __init__(self, max_memory_gb=4):
        self._models: Dict[str, any] = {}
        self._model_usage: Dict[str, dict] = {}
        self._max_memory = max_memory_gb * 1024 * 1024 * 1024  # GB to bytes
        self._lock = threading.Lock()

    def load_model(self, model_type: str, model_name: str, loader_func):
        """加载模型（带内存检查）"""
        with self._lock:
            cache_key = f"{model_type}_{model_name}"

            # 检查是否已加载
            if cache_key in self._models:
                self._model_usage[cache_key]['last_used'] = time.time()
                return self._models[cache_key]

            # 检查内存使用
            current_memory = psutil.Process().memory_info().rss
            if current_memory > self._max_memory * 0.8:  # 80% 阈值
                # 释放最久未使用的模型
                self._release_oldest_model()

            # 加载新模型
            model = loader_func(model_name)
            self._models[cache_key] = model
            self._model_usage[cache_key] = {
                'loaded_at': time.time(),
                'last_used': time.time(),
                'memory_estimate': self._estimate_model_size(model)
            }

            return model

    def release_model(self, model_type: str, model_name: str):
        """释放指定模型"""
        cache_key = f"{model_type}_{model_name}"
        with self._lock:
            if cache_key in self._models:
                del self._models[cache_key]
                del self._model_usage[cache_key]
                # 清理 GPU 内存
                if torch.cuda.is_available():
                    torch.cuda.empty_cache()

    def _release_oldest_model(self):
        """释放最久未使用的模型"""
        if not self._model_usage:
            return

        oldest_key = min(
            self._model_usage.keys(),
            key=lambda k: self._model_usage[k]['last_used']
        )

        del self._models[oldest_key]
        del self._model_usage[oldest_key]

        if torch.cuda.is_available():
            torch.cuda.empty_cache()

    def _estimate_model_size(self, model):
        """估算模型内存占用"""
        # 简化估算：参数数量 * 4 bytes (float32)
        if hasattr(model, 'parameters'):
            param_size = sum(p.numel() * 4 for p in model.parameters())
            return param_size
        return 100 * 1024 * 1024  # 默认 100MB

    def get_memory_stats(self):
        """获取内存统计"""
        current_memory = psutil.Process().memory_info().rss
        return {
            "current_memory_mb": current_memory / 1024 / 1024,
            "max_memory_mb": self._max_memory / 1024 / 1024,
            "loaded_models": len(self._models),
            "models": list(self._model_usage.keys())
        }

# 全局模型管理器
model_manager = ModelManager(max_memory_gb=4)
```
